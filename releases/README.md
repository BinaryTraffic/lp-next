# 版の固定点（`releases/`）

**過去版の参照元は、ここにコピーしたスナップショットではなく、Git タグです。**

| タグ | 指す内容（概要） |
|------|------------------|
| **v1.0.0** | 初回コミット相当のツリー。レイアウトはリポジトリ直下。 |
| **v1.1.0** | v1.1.x 安定ライン（1.1.11 系）。 |
| **v1.2.0** | v1.2 系・`current/` + リポジトリ構成整備を含めた本番用安定系。 |

`git archive` の例（リポジトリルートで）:

```bash
git archive --format=tar v1.1.0 | tar -t | head
```

作業用にチェックアウトする例:

```bash
git worktree add -b work-v1.1.0 ../lp-next-v1.1.0 v1.1.0
```

`v1.0.0` や `v1.1.0` の時点では **`current/` 配下**ではなく、ルートの `lp_reverse_cms/` など従来レイアウトの場合があります。`v1.2.0` 以降のメイン作業基準は **`current/`** です。

**レガシー:** ドキュ上の揃いのため **`v1.1.11-stable`** タグが別に残る場合があります。意味の固定点は **`v1.1.0`** を優先してください。
