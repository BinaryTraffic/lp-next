LP-NEXT（LP Reverse CMS）

このページは リポジトリルートを DocumentRoot にしたとき の current/ 入口です。管理画面は lp_reverse_cms/ 配下にあります。リポジトリは GCP の VM などに SSH して clone した作業ディレクトリ上に置かれ、同じホスト（または前面のロードバランサ）が HTTPS で配信する、という構成を想定しています（手元 PC のフォルダを直接 DocumentRoot にしているわけではありません）。

プロジェクトジャーナル（v1.2.0 起点）

文頭にフェーズ進捗、打ち合わせメモ（端末内保存・JSON 受け渡し）を置いています。スマホは表を横にスクロール。

  • journal.html — 整形表示（推奨）
  • JOURNAL.md（同じ文面）
  • prompt_context_demo.html — 店舗文脈（会社→事業→地域流用）ステップ式テスト
  • lp_layout_images.html — 解析済み LP のレイアウト画像リスト（サムネ＋再読込）
  • lp_ai_image_review.html — クローン画像 × 店舗文脈JSONプロンプトのAI差し替え検証
管理画面を開く
  • lp_reverse_cms/（index.php、アプリ v1.2.0）

ドキュメント（経緯・構築手順はこの HTML のみで十分です）

  • lp_reverse_cms/docs/PROJECT_HISTORY_AND_SETUP.html（経緯・ゼロからの構築）
  • lp_reverse_cms/docs/（上と同じ内容を開きます）

その他

  • README.md（ルート・リポジトリ概要）
  • lp_reverse_cms/README.md（アプリ概要・起動）
  • ENVIRONMENT_AND_OPERATIONS.md（環境・運用）

DocumentRoot を lp_reverse_cms だけにしている場合は、この index.html は URL からは参照されません。直接 管理画面の URL を開いてください。
